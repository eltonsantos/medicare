## ğŸš€ Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/eltonsantos/medicare.git && cd medicare
```

Follow the steps below

```bash
# Install the dependencies
$ bundle install

# Create database
$ rails db:create

# Run migrations
$ rails db:migrate

# Run seed
$ rails db:seed

# Start the project
$ bin/dev
```
